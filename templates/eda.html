EDA (Análisis exploratorio de datos)

Contexto

Se enfoca en tener una idea de la estructura del conjunto de datos, identificar la variable objetivo y posibles técnicas de modelado. Es útil también revisar la descripción de los datos para comprender lo que significa cada característica. EDA permite identificar las técnicas estadísticas correctas tanto para analizar como para pronosticar los datos. Se dividen en dos tipos básicos de técnicas:

  • Técnicas gráficas.
  • Técnicas cuantitativas.
Para las técnicas gráficas se busca mostrar las propiedades clave dentro de un conjunto de datos, con un formato que convenga dicho análisis. Se utilizan diversas técnicas gráficas, entre las más utilizadas y recomendadas se encuentran los diagramas de caja y los histogramas. Para las técnicas cuantitativas se tiene un método un poco más riguroso para la obtención de las propiedades clave.
Entre los pasos principales se encuentran:
  • Paso 1: Descripción de la estructura de los datos.
  • Paso 2: Identificación de datos faltantes.
  • Paso 3: Detección de valores atípicos.
  • Paso 4: Identificación de relaciones entre pares variables.
