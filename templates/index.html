¿Qué es la mineria de datos?
Objetivo del proyecto

Este es el proyecto para la clase de Minería de Datos para el semestre 2021-2. El objetivo de este proyecto es mostrar algunos Algoritmos utilizados en esta materia asi como una implementación que se podrá usar en linea.

Algoritmos implementados:

  • EDA.
  • Selección de Características.
  • Métricas.
  • PCA.
  • Cluster Particional (K-Means).
  • Reglas de asociación (Apriori).
  • Regresión Lineal.
