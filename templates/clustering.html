Clústering

Contexto

El clustering es una tarea que tiene como finalidad principal lograr el agrupamiento de conjuntos de objetos no etiquetados, para lograr construir subconjuntos de datos conocidos como Clusters.

En este proyecto se presentan dos tipos de clustering:

  • Clusterin particional.
  • Clustering jerárquico.
El clustering particional se enfoca en obtener una partición de elementos por grupo o en su defecto clusters, con el fin de que cada elemento pertenezca a uno de los clusters y que además cada cluster sea disjunto.
El clustering jerárquico es una técnica que regularmente es usado como método de minado de datos y consiste en la agrupación de datos, donde a cada agrupación se le conoce como clúster. El algoritmo de clustering jerárquico, asigna las agrupaciones de acuerdo a las distancias que existen por cada elemento, entre estas se puede ejecutar diversos métodos para el cálculo de distancia, entre los más ocupados se encuentra la métrica euclidiana.
